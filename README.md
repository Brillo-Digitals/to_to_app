✅ To-Do App

A simple and efficient task management application built to help users organize daily activities, track productivity, and stay focused.

This project demonstrates clean UI design, structured state management, and practical data handling across modern development stacks.

📌 Overview

Staying productive starts with clarity. The To-Do App allows users to create, manage, and complete tasks in a simple and distraction-free environment.

Designed with usability in mind, the app focuses on speed, simplicity, and efficiency.

✨ Features

Add new tasks

Edit existing tasks

Mark tasks as completed

Delete tasks

Real-time task updates

Clean and responsive UI

Persistent storage 

🏗 Tech Stack

Flutter – Mobile application

Local storage / Hive

🚀 Getting Started
Clone the Repository
```Bash
git https://github.com/Brillo-Digitals/to_to_app.git
```
```Bash
cd todo-app
```
Run Flutter Version
```Dart
flutter pub get
flutter run
```
